# Reviews

This directory stores design, code and business reviews.

## Purpose
Every significant feature should be reviewed before being considered complete.

## Recommended structure
- Summary
- Strengths
- Weaknesses
- Risks
- Required improvements
- Final decision

## Review Types
- Product Review
- UX Review
- Architecture Review
- Code Review
- Business Review
- AI Review

Reviews become part of the permanent project knowledge and should be referenced by future work when relevant.
